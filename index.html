YoioJava

RESTful Web Services in Java.

关于

YoioJava Web 框架是一款开源产品, 为开发者带来高质量，大道至简的开发体验。

YoioJava旨在帮助开发者团队开发简洁的，可移植的，快速灵活的基于JVM的系统和应用。 YoioJava提供了丰富的 API 使开发者可以快速上手，YoioJava是简洁的，接口化的。

开发者不仅可以基于YoioJava开发商业应用，也暴露许多扩展spi,这样开发人员可以使用YoioJava扩展最适合他们的需求。

YoioJava项目的目标可以概括为以下几点:

  • 大道至简！
  • 提供可扩展的API & 建立用户和开发者的交流社区; 最后，
  • 更容易，更简单的使用Java平台快速构建WEB应用及服务.
