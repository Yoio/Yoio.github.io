该章节介绍如何获取YoioJava源代码或者二进制jar包.

1.1. 获取YoioJava源代码

YoioJava 项目源代码托管在 GitHub上面。

Note

你可以通过YoioJava的GitHub项目主页直接下载源代码。

https://github.com/Yoio/YoioJava

点击Download Zip，下载源文件。

1.2. 获取YoioJava二进制jar包

最新的二进制jar包也发布在GitHub上面，你有进入YoioJava项目主页，点击releases查看YoioJava发布的最新版本。

Note

你可以通过YoioJava的GitHub项目主页直接下载源代码。

https://github.com/Yoio/YoioJava/releases
